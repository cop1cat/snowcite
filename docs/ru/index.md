# snowcite

MCP-сервер для проведения систематических литературных обзоров. Выполняет поиск
по arXiv, Semantic Scholar, OpenAlex, Crossref и PubMed; развивает библиографию
через граф цитирований; проводит ревью статей в чате с Claude; генерирует
документы на LaTeX или Typst и компилирует их в PDF.

!!! info "Без веб-интерфейса"
    Ревью проходит в чате. Claude обрабатывает абстракты батчами, самостоятельно
    фильтрует очевидные случаи и выносит на обсуждение только пограничные. Это
    архитектурное решение, подробнее см. раздел [Workflow](workflow.md).

## Ключевые особенности

- **Поэтапная подготовка текста.** План документа, затем короткий черновик
  разделов, затем их развёрнутые версии с проверкой соответствия плану. Работа
  не сваливается на автора одним блоком на две с половиной тысячи слов.
- **Два независимых субагента для ревью.** Академический ревьюер проверяет
  утверждения и ссылки; языковой — читаемость текста. Каждый запускается
  с чистым контекстом и даёт независимую оценку.
- **Проект — это директория.** Каталог `.snowcite/` хранится рядом с
  исходниками, по аналогии с `.git`. Переключение между проектами — `cd`,
  перенос на другую машину — `git clone` или копирование каталога.
- **Typst как основной бэкенд.** Поддержка кириллицы работает без
  дополнительных настроек. LaTeX доступен для случаев, когда требуется
  строгое соответствие ГОСТу или вузовский шаблон.
- **Мультидисциплинарность.** Набор источников подбирается автоматически
  исходя из области: arXiv включается для STEM-направлений, PubMed — для
  медицины и биологии, Semantic Scholar, OpenAlex и Crossref используются
  во всех случаях.
